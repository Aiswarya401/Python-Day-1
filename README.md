# Python-Day-1
📌 Overview

This is a beginner-friendly Python project built on Day 1 of my Python learning journey.
The program generates a random number, and the user has to guess it with hints provided after each attempt.

🚀 Features
Random number generation 🎲
User input handling
Smart hints:
🔥 Very Close
🙂 Close
❄️ Far
Attempt counter
Loop-based gameplay

🧠 Concepts Used
Variables
Input / Output
Conditional Statements (if-else)
Loops (while)
Functions (basic usage)
Python random module

💻 How to Run
Clone the repository:
git clone https://github.com/your-username/python-day1-number-guessing-game.git
Run the program:
python number_guessing_game.py
